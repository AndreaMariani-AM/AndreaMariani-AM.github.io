PBPnet

PBPnet extends the bpnet-lite framework with Polycomb-specific modifications to model how Polycomb proteins bind along genomic sequences. I built custom PyTorch DataLoaders for training and validation, added concurrent training support, and extended the genomic context to 5kb for finer signal resolution. To work within GPU memory constraints, I wrote a preprocessing pipeline that extracts sequences and binding signals from reference genomes and stores them in HDF5 format — making large genomic datasets tractable without on-the-fly generation. The result is a sequence-to-signal model that helps decode the regulatory logic behind Polycomb-mediated chromatin organization.

agent_autoresearch

agent_autoresearch is an autonomous experiment loop for optimizing whole-slide image (WSI) classification in computational pathology, built around a hill-climbing strategy driven by Claude Code. Parallel subagents propose, implement, and evaluate architectural changes to a Multiple Instance Learning model — encoder design, aggregation strategies, loss functions, training schedules — retaining improvements and discarding dead ends automatically. Starting from a baseline of 0.768 macro AUROC, the system reached 0.852 through iterative self-improvement, all without manual intervention between runs. The project sits at the intersection of AI-assisted research and computational pathology, built on PyTorch Lightning with Virchow2 patch embeddings as the foundation model.
