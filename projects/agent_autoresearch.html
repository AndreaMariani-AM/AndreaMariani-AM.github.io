agent_autoresearch

agent_autoresearch is an autonomous experiment loop for optimizing whole-slide image (WSI) classification in computational pathology, built around a hill-climbing strategy driven by Claude Code. Parallel subagents propose, implement, and evaluate architectural changes to a Multiple Instance Learning model — retaining improvements and discarding dead ends automatically.

Starting from a baseline of 0.768 macro AUROC, the system reached 0.852 through iterative self-improvement, all without manual intervention between runs. Modifications span encoder design, aggregation strategies, loss functions, and training schedules.

The project sits at the intersection of AI-assisted research and computational pathology, built on PyTorch Lightning with Virchow2 patch embeddings as the foundation model.
