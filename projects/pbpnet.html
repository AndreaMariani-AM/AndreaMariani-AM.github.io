PBPnet

PBPnet extends the bpnet-lite framework with Polycomb-specific modifications to model how Polycomb proteins bind along genomic sequences. I built custom PyTorch DataLoaders for training and validation, added concurrent training support, and extended the genomic context to 5kb for finer signal resolution.

To work within GPU memory constraints, I wrote a preprocessing pipeline that extracts sequences and binding signals from reference genomes and stores them in HDF5 format — making large genomic datasets tractable without on-the-fly generation.

The result is a sequence-to-function model that helps decode the regulatory logic behind Polycomb-mediated chromatin organization.
